What is Time-Sensitive Networking?

Time-Sensitive Networking (TSN) is a set of IEEE 802.1 standards that transforms standard Ethernet into a deterministic, real-time communication platform. TSN ensures guaranteed packet transport with bounded latency, low packet delay variation, and minimal packet loss.

üéØ The Core Mission

TSN provides deterministic connectivity through IEEE 802 networks, enabling time-critical and best-effort traffic to coexist on the same infrastructure while meeting strict timing requirements.

IEEE 802.1 TSN Standards

TSN is not a single protocol but a collection of IEEE 802.1 standards that work together to provide deterministic communication. Here are the key standards:

IEEE 802.1Q-2022

Bridges and Bridged Networks - The foundational standard defining virtual LANs and network switches with TSN extensions.

IEEE 802.1AS-2020

Timing and Synchronization for Time-Sensitive Applications - Provides time synchronization ensuring all devices share the same sense of time, which is fundamental for determinism.

IEEE 802.1Qbv

Enhancements for Scheduled Traffic - Defines "time-aware shapers" that assign specific time slots to different traffic types based on priority.

IEEE 802.1Qbu

Frame Preemption - Allows high-priority frames to interrupt lower-priority frame transmission, reducing latency for critical data.

IEEE 802.1CB-2017

Frame Replication and Elimination for Reliability - Enhances network reliability by replicating frames and eliminating duplicates.

IEEE 802.1Qca

Path Control and Reservation - Manages network paths and reserves bandwidth for specific traffic streams.

IEEE 802.1Qcc

Stream Reservation Protocol (SRP) Enhancements - Improves stream reservation and performance for better resource management.

IEEE 802.1Qch

Cyclic Queuing and Forwarding - Enables deterministic forwarding of frames through cyclic scheduling.

IEEE 802.1Qci

Per-Stream Filtering and Policing - Provides security and quality control by filtering and policing individual data streams.

IEEE 802.3br

Interspersing Express Traffic - Physical layer support for frame preemption at the MAC layer.

üìö Base Standards

TSN builds upon several foundational IEEE standards including 802.1AB (LLDP), 802.1AS (Timing), 802.1AX (Link Aggregation), and 802.1CS (Link-local Registration Protocol).
